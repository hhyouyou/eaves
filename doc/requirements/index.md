## 如何提新需求

1. 复制 [template-requirement.md](template-requirement.md)。
2. 重命名为 `REQ-编号-需求名称.md`。
3. 按模板填写内容。
4. 在本文件对应状态列表中登记。

## 如何提 Bug

1. 复制 [template-bug.md](template-bug.md)。
2. 重命名为 `BUG-编号-问题描述.md`。
3. 按模板填写内容。
4. 在本文件 Bug 列表中登记。
