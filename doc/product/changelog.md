## v0.2.0 —— 国家统计局70城房价数据采集模块（2026-06-22）

```mermaid
graph LR
    A[接口配置封装] --> B[HTTP客户端]
    B --> C[ODS/DWD/DIM数据加载]
    C --> D[DWS/ADS汇总分析]
    D --> E[定时任务入口]
```

### 新增

- **[数据采集模块]**：新增 `backend/data_collection/` 模块，支持70城住宅销售价格数据采集。
- **[HTTP客户端]**：`client.py` 封装国家统计局两个接口，内置1-5秒随机风控间隔。
- **[数据加载器]**：`loader.py` 实现 ODS 原始数据、DWD 明细数据、DIM 维度数据写入。
- **[数据汇总器]**：`aggregator.py` 实现 DWS 月度汇总、趋势特征、ADS 分析报告生成。
- **[采集运行器]**：`runner.py` 支持全量初始化、单城市补采、月度增量更新三种模式。
- **[定时任务入口]**：`scheduler.py` 提供定时任务调用入口，便于后续配置每月自动更新。
- **[南京数据初始化]**：完成南京市2020年1月至2026年5月共77个月数据入库。

### 优化

- 城市间采集间隔调整为3-5分钟，进一步降低触发反爬风险。
- 按年份分段请求数据，提升采集稳定性。

### 修复

- 无

## 版本号规则

- **v0.N.0**：每次迭代或发布一个阶段性功能时递增 N。
- **v0.N.x**：仅修复小 Bug 或做细微优化时递增 x。
- 进入正式运营后可考虑使用 **v1.0.0**。
